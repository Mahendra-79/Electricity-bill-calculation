# Electricity-bill-calculation
A C-language base project named elctricity bill calculations.It mainly works on the electricity bill calculations depending upon the how many units of current consumed by the user.

 Features of Electricity Bill Management System
 
1 Bill Calculation

The program automatically calculates the electricity bill based on the number of units used.
It applies different rates for different unit ranges:

Up to 100 units → ₹5 per unit

From 101 to 200 units → ₹7 per unit

Above 200 units → ₹10 per unit (optional higher range for large usage)

2️ Search Customer Details

You can search for any customer by entering their ID number.
If the record is found, the system shows all details of that customer.

3️  Update Customer Information

If any customer details need to be changed, like name or units, the system allows you to update them easily.
After updating, the bill amount is recalculated automatically.

4️ Delete Customer Record

If a record is not needed, it can be deleted from the list.
After deleting, the remaining records are shifted properly to keep the data in order.

5️  Menu-Driven System

The program uses a simple text-based menu.
It allows users to choose different operations like Add, Display, Search, Update, Delete, or Exit.

6️ Continuous Program Execution

The program keeps running until the user chooses the Exit option.
This helps perform many operations in one session without restarting.
